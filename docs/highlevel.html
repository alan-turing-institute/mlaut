class mlaut.highlevel.tasks.BaseTask(target, features=None, metadata=None)[source]¶

Bases: object

Abstract base task class. A task encapsulates metadata information such as the feature and target variable which to fit the data to and additional necessary instructions on how to fit and predict. Implements attributes and operations shared by all tasks, including compatibility checks between the concrete task type and passed metadata. :param target: The column name for the target variable to be predicted. :type target: str :param features: The column name(s) for the feature variable. If None, every column apart from target will be used as a feature. :type features: list of str, optinal, (default=None) :param metadata: Contains the metadata that the task is expected to work with. :type metadata: pandas.DataFrame
